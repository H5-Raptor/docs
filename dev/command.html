# 介绍

raptor不干预业务开发，不要求业务必须使用指定组件开发环境。

但提供了一个组件开发环境供参考，命令式操作。

支持组件从创建、调试、多语言解析、编译到发布的完整生命周期。

# 创建组件

npm run create ${pathName}

# 调试组件

npm run test

# 多语言解析

npm run xlsx

# 编译组件

npm run build

# 发布组件

npm run publish

提示

作为业务组内第一个了解本中台的你，强烈建议将使用文档完整过目一遍。
