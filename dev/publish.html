# 组件发布

组件发布到中台有 2 种途径，分别是：

  1. 中台 Web 页面填写表单发布。
  2. 使用开发者 token，调用 Open API 发布。

# 表单发布

注意事项：

  1. 如果是第一次发布，组件名要唯一。
  2. 如果是组件迭代，版本号需要往上提升。
