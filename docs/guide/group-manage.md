# 组管理

## 新建组
通过新建组按钮，填写组名即可新建组，创建者即为该组组长。

## 拉拢组员
批量添加账号为该组组员、组长。

::: tip
组与组员、组长是多对多关系。
组可以含有多个组员、组长，一个账号可以处于多个组中。
:::

## 组件类型
为本组添加组件类型，可从功能、业务角度考虑命名。
## 预览地址
预览页用于2处
1. 编辑时的预览区，目的是方便编辑者实时预览
2. 独立的预览页，目的用于手机端访问
## 预览amd配置
预览页基于requirejs的amd环境，组件编译时不将依赖打包进组件，而是会在组件代码中形成import或require依赖。
